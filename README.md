# TANEI 计算机社团示例页面

这是一个简约、配色不过于鲜艳的静态社团网页示例，文件结构：

- `index.html` - 页面主文件（中文内容，语义化标签）
- `styles.css` - 样式文件，使用 CSS 变量与响应式网格

如何使用：

1. 在本地打开 `index.html` 文件（双击或在浏览器中打开）。
2. 若需修改文字或章节，编辑 `index.html`。
3. 若需调整配色或排版，编辑 `styles.css` 中的 :root 色值与变量。

设计说明：

- 采用柔和青蓝作为强调色，主背景为浅灰，避免过渡鲜艳的配色。
- 使用简约卡片与圆形头像以提升识别度。

你可以把该项目放到静态主机（例如 GitHub Pages）上直接发布。
